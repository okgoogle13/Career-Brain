# Enhanced Resume Agent 3.0 – Personal Edition

Welcome! This repository contains the **personal, single-user** version of the Enhanced Resume Agent 3.0, now equipped with a powerful background automation suite. All code, configuration, and documentation have been streamlined for an individual running the system on their own computer or personal Google Cloud account.

## ✨ Core Features

-   **AI-Powered Optimization:** Tailors resumes, cover letters, and KSC responses using Google Gemini.
-   **Autonomous Job Scouting:** Automatically checks your Gmail for new job postings, parses them, and prepares draft documents.
-   **Automated Workflow Management:** Uses Google Drive to store documents, Google Calendar to set follow-up reminders, and Google Tasks to manage your application pipeline.
-   **Simple Interface:** A lightweight, custom HTML/JS frontend served directly by the backend.
-   **Easy Deployment:** Designed for one-click setup in Google AI Studio or deployment to a personal Google Cloud Run account.

## 🚀 Quick Start

### Prerequisites
-   Python 3.11+
-   Docker (Recommended) or Google AI Studio
-   Your personal API keys for Google Services (Gmail, Drive, etc.) and LLMs.

### Setup & Running

This project is designed to be run in a multi-process environment like Google AI Studio or with Docker Compose.

1.  **Configure Your Environment:** Copy `.env.example` to `.env` and fill in your API keys.
2.  **Run with Docker Compose:**
    ```bash
    docker-compose up --build
    ```
3.  **Access the Services:**
    -   **Application UI:** `http://localhost:8000`
    -   **API Docs:** `http://localhost:8000/api/docs`
    -   **Redis Broker:** `localhost:6379`
