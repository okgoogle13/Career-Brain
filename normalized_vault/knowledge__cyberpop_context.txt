# **CareerCopilot Design System: Authentic Intelligence**

This document contains the complete design system for the CareerCopilot application, based on the "Authentic Intelligence" theme. It is the single source of truth for all visual design, typographic rules, and color tokens.

## **1\. Mood Board: Authentic Intelligence**

🔮 **Mood Board: Authentic Intelligence**

**Keywords:** Professional • Structured • Depth-Aware • Humanist • Creative

**1\. The Vibe (Visual Metaphor)**

* **Concept:** "A high-end, modern workspace tool (like Linear or Notion) but with the warmth and emotive readability of a printed magazine. This is 'Grounded AI'—trustworthy, structured, and substantial."  
* **Key Feelings:** Creative, friendly, intelligent, structured, and authentic.  
* **Lighting:** High-contrast and clean. It is not "glowing neon"; it is "lit from within."

**2\. The Palette (Cyber-Pop Colors)**

* **The Canvas (Surfaces):**  
  * \#141318 (Surface) – **Near-Black.** The main app background.  
  * \#201F25 (Surface Container) – **Dark Grey.** The default card/panel color.  
* **The Accents (Your P/P/G Palette):**  
  * \#C4BFFF (Primary) – **Vibrant Purple.** Creative and professional.  
  * \#56DBBE (Secondary) – **Electric Mint.** Energetic and positive.  
  * \#FFAEDC (Tertiary) – **Bright Pink.** Playful, surprising, "magic" accent.  
* **The Text (Content):**  
  * \#E6E1E5 (On Surface) – **Light Grey.** The default body text.

3\. Typography (The "Humanist" System)  
This system is built on a dual-font pairing to separate Interface from Content.

* **The "Structured" UI Font: Roboto Flex**  
  * **Feeling:** Precision, efficiency, clean, modern.  
  * **Usage:** All UI chrome: Buttons, navigation, labels, tabs, card titles.  
* **The "Humanist" Content Font: Roboto Serif**  
  * **Feeling:** Authentic, friendly, emotive, and trustworthy.  
  * **Usage:** All long-form generated content: AI responses, resume analysis, cover letter drafts. This makes the AI's voice feel more human.

**4\. Materiality & Texture (The "Stacked Paper" Metaphor)**

* **Surface:** Matte, velvety dark surfaces.  
* **Depth:** 100% Tonal. We **do not use shadows**. Depth is communicated through color tone shifts (lighter \= higher).  
  * **Layer 0 (Desk):** \#141318 (Surface)  
  * **Layer 1 (Sidebar):** \#1C1B20 (Surface Container Low)  
  * **Layer 2 (Canvas):** \#201F25 (Surface Container)  
  * **Layer 3 (Modal):** \#2B2A2F (Surface Container High)  
* **Borders:** Subtle 1px outlines (\#938F99).

**5\. UI Pattern Inspiration**

* **"Playful" AI:** Use the **Dark Pink** (\#712F62) background for the AI chat window. Inside it, use the **Light Pink** (\#FFD7EE) *text color* and, most importantly, the Roboto Serif font.  
* **"Structured" Hierarchy:** The UI is clean, sharp, and grid-aligned, built with Roboto Flex.  
* **"Organic" Content:** The AI's output is emotive and readable, set in Roboto Serif with a generous line height.

## **2\. Project Style Guide (M3\_GUIDE.md)**

Project Style Guide: CareerCopilot  
System: Material 3 (Strict 26-Role Implementation)  
Theme: Authentic Intelligence (Cyber-Pop Palette)  
Target Platform: Desktop Web (Expanded Screens)  
1\. Typography  
Strategy: High-legibility mechanical UI paired with warm, editorial readability.

* **A. Headings & UI Components (Buttons, Nav, Inputs)**  
  * **Font:** Roboto Flex  
  * **Usage:** App Bars, Navigation Rails, Button text, Input labels, Data Tables, Card Titles.  
  * **Features:** Utilize the wdth (Width) and GRAD (Grade) axes to fine-tune density on desktop screens.  
  * **Instance: ui-emphasis**  
    * font-weight: 600;  
    * font-variation-settings: "'GRAD' 50";  
* **B. Body Copy & Long-Form Content**  
  * **Font:** Roboto Serif  
  * **Usage:** User profile bios, generated AI reports, documentation blocks, help text.  
  * **Rationale:** Adds a "genuine" and authentic tone to the AI's voice, differentiating content from UI.  
  * **Instance: content-body**  
    * font-family: 'Roboto Serif', serif;  
    * font-weight: 400;  
    * line-height: 1.6;

2\. Color System (The 26-Role Schema)  
Core Rule: Adherence to contrast pairs from material-theme-cyberpop.json is mandatory.  
**A. The Surface Hierarchy (Desktop/Expanded Strategy)** *Because this is a desktop app, we rely on depth (tonal elevation) to separate regions. A flat black screen is forbidden.*

| Role | Token Name | Hex (Dark) | Usage |
| :---- | :---- | :---- | :---- |
| **Level 0** | surface | \#141318 | The fundamental background of the window. |
| **Level 1** | surfaceContainerLow | \#1C1B20 | Navigation Rail or Sidebar (recedes). |
| **Level 2** | surfaceContainer | \#201F25 | **Default** fill for main content cards. |
| **Level 3** | surfaceContainerHigh | \#2B2A2F | Modals, Dialogs, and Hover states. |

**B. Accent Colors**

| Role | Token Name | Hex (Dark) | Usage |
| :---- | :---- | :---- | :---- |
| **Primary** | primary / onPrimary | \#C4BFFF / \#412DAB | Key actions (Submit, Save, "Generate"). |
| **Active Nav** | primaryContainer | \#5844C3 | Active pill in sidebar. |
| **Secondary** | secondary / onSecondary | \#56DBBE / \#00382E | Tonal actions (Filters, Chips, Success). |
| **Tertiary** | tertiaryContainer | \#712F62 | "AI Insights" highlights (Dark Pink). |
| **On Tertiary** | onTertiaryContainer | \#FFD7EE | Text *on* the AI highlight (Light Pink). |

**C. Feedback & Boundaries**

| Role | Token Name | Hex (Dark) | Usage |
| :---- | :---- | :---- | :---- |
| **Error** | error / onError | \#FFB4AB / \#690005 | Critical failures. |
| **Input Border** | outline | \#938F99 | Input borders, Checkboxes (3:1 contrast). |
| **Divider** | outlineVariant | \#48454E | Decorative dividers (Low contrast). |

**3\. Accessibility & Implementation Rules**

* **Never Guess Pairs:** Always use the "On" variant for text (e.g., Text on primary background *must* be onPrimary).  
* **Desktop Density:** Use Surface Container roles to define hierarchy.  
* **Type Hierarchy:** Do not mix Roboto Flex and Roboto Serif on the same line. Keep UI (Flex) distinct from Content (Serif).

**4\. Component Guidelines**

* **Cards (Standard)**  
  * **Background:** surfaceContainer (\#201F25)  
  * **Border:** 1px solid outline (\#938F99)  
  * **Radius:** Large (16px)  
  * **Title Font:** Roboto Flex (use ui-emphasis instance)  
  * **Body Font:** Roboto Serif (use content-body instance)  
* **Buttons (Primary)**  
  * **Background:** primary (\#C4BFFF)  
  * **Text:** onPrimary (\#412DAB).  
  * **Font:** Roboto Flex (use ui-emphasis instance)  
* **AI Chat / "Magic" Component**  
  * **Background:** tertiaryContainer (\#712F62 \- Dark Pink)  
  * **Text:** onTertiaryContainer (\#FFD7EE \- Light Pink)  
  * **Radius:** Large (16px)  
  * **Title Font:** Roboto Flex  
  * **Body Font:** Roboto Serif (to make AI voice authentic)  
* **Navigation Rail (Sidebar)**  
  * **Background:** surfaceContainerLow (\#1C1B20)  
  * **Font:** Roboto Flex
