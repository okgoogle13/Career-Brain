# Handover Prompt

This bundle contains the final schema, batch-generated theme JSONs, planning prompts, execution prompts, and supporting docs for the ATS theme system refresh.

Use this package to:
1. Validate the master schema against the batch-updated theme files.
2. Regenerate any stale theme JSON using schema v2.3 behavior.
3. Preserve ATS-safe constraints and single-column layout rules.
4. Keep each theme explicitly distinct in silhouette, band strategy, divider grammar, accent placement, and density target.
5. Avoid tonal-only or palette-only variation.

Recommended next action:
- Review the master schema and then continue with the batch update workflow if any downstream files still use the older structure.
